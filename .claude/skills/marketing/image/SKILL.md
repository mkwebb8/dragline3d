# Image Creation

Create professional marketing images using AI generation and design tools.

## Tool Selection

| Tool | Best For |
|------|----------|
| Gemini / Nano Banana | All-purpose; good text rendering |
| Flux | Photorealism, brand consistency |
| Ideogram 3.0 | Typography and text in images |
| ChatGPT Images | Quick generation, varied styles |
| Midjourney | High-quality artistic imagery |
| Recraft | Vector-style, brand assets |
| Canva | Templated graphics, fast variants |
| Figma | Design systems, pixel-perfect control |

## Standard Dimensions

| Image Type | Dimensions |
|-----------|-----------|
| Blog hero | 1200 × 630px |
| OG image | 1200 × 630px |
| Twitter/X card | 1200 × 628px |
| LinkedIn cover | 1128 × 191px |
| Instagram square | 1080 × 1080px |
| Instagram story | 1080 × 1920px |

## Format Guidance

- **WebP**: Preferred for web (smaller file size, same quality)
- **JPEG**: Fallback for broad compatibility
- **PNG**: For images with transparency
- **SVG**: For logos and icons (scales without loss)

## Required Inputs

1. Image type (hero, social, mockup, banner, brand asset)
2. Platform and dimensions
3. Brand context (colors, fonts, style)
4. One-off or repeatable template?
5. Tools available / API keys?

## Key Principle

For product screenshots: use real UI, not AI hallucinations. AI-generated "screenshots" look fake and damage credibility.

## Integration

- social: social media image dimensions and content
- ad-creative: image specs per ad platform
- copywriting: image and copy work together for conversion

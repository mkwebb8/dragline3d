# Onboarding CRO

Optimize the post-signup experience to get users to their aha moment quickly. Time-to-Value is everything.

## Core Principle

Do over show. Remove friction between signup and experiencing core value. One focused goal per session, not a feature tour.

## Activation Definition

First: identify the specific action that correlates with retention (your activation event). Examples:
- Created first project
- Connected first integration
- Invited first team member
- Completed first transaction

Everything else is in service of reaching that event faster.

## Three Flow Approaches

**Product-first**: Drop user directly into the product. Best for simple, intuitive products.

**Guided setup**: Step-by-step wizard that configures the product. Best for B2B tools requiring setup to deliver value.

**Value-first**: Show a demo or filled-in example before asking for setup effort. Best for complex products where empty states feel like a blank page.

## Key Design Elements

- **Progress indicators**: Show where they are in setup (reduces abandonment)
- **Empty states**: Fill them with templates, examples, or guiding actions — never leave them blank
- **Checklists**: One action per item; check off on completion; show % complete
- **Tooltips**: Contextual, triggered by user action — not a feature tour

## Multi-Channel Coordination

In-app and email reinforce each other without duplicating:
- In-app: what to do right now
- Email: why to come back + what to do next

## Measurement

- Activation rate (% reaching activation event within 7 days)
- Time to activation (median hours)
- Onboarding completion rate
- Day-1, Day-7, Day-30 retention by cohort

## Integration

- signup: optimizing the step before onboarding starts
- emails: onboarding email sequence coordination
- churn-prevention: poor onboarding predicts churn
- ab-testing: testing onboarding flow variations

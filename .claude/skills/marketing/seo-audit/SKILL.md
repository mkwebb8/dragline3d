# SEO Audit

Comprehensive SEO assessment across technical, on-page, content quality, and international dimensions. Output: prioritized audit report with specific fixes and impact assessments.

## What Gets Evaluated

**Technical SEO**: Crawlability, indexation, site speed, mobile-friendliness, security, URL structure

**On-Page Optimization**: Title tags, meta descriptions, heading hierarchy, content depth, internal linking

**Content Quality**: E-E-A-T signals (Experience, Expertise, Authoritativeness, Trustworthiness), topical authority, user engagement indicators

**International/Localization**: hreflang setup, canonicalization, locale structure, translation quality

**Common Problem Areas**: Specific to site type (SaaS, e-commerce, blog, local business)

## Required Inputs

1. Site type (SaaS, e-commerce, blog, local business)
2. Business goals — what keywords/topics matter most?
3. Current state — known issues, traffic changes, recent migrations?
4. Scope — full audit or specific pages/problems?
5. Access — Google Search Console or analytics data available?

## Key Tool

For schema markup validation, use Google Rich Results Test (renders JavaScript) rather than static HTML analysis alone.

## Integration

- ai-seo: optimizing for AI citation and AI search visibility
- schema: structured data implementation
- site-architecture: URL structure and internal linking
- programmatic-seo: at-scale page creation
- analytics: measuring SEO impact

# App Store Optimization (ASO)

Analyze and optimize App Store and Google Play listings for search ranking and download conversion.

## Scoring Dimensions

| Dimension | Weight | What's Evaluated |
|-----------|--------|-----------------|
| Title / Subtitle | 20% | Keyword placement, readability |
| Description | 15% | First 2 lines (above fold), keyword density |
| Visuals | 25% | Screenshots, preview video, icon |
| Ratings & Reviews | 20% | Average rating, review volume, recency, responses |
| Metadata | 10% | Keywords field (Apple), category selection |
| Conversion signals | 10% | Social proof, feature highlights, store presence |

## Platform Differences

**Apple App Store:**
- Indexes: title (30 chars), subtitle (30 chars), keyword field (100 bytes)
- Screenshot captions are searchable
- Long description: conversion only (not indexed)

**Google Play:**
- Indexes: title, short description (80 chars), full description (4,000 chars)
- No hidden keyword field — use keywords in description
- Feature graphic required for featured placement opportunities

## Brand Maturity Scoring

Apply different standards based on brand tier:
- **Dominant brands**: Deliberate brand choices assessed differently from optimization gaps
- **Established leaders**: Competitive benchmark comparison
- **Challengers**: Full optimization recommendations

## Output

- Scored audit (0–100) with letter grade
- Specific, implementable recommendations prioritized by impact-to-effort
- Competitor comparison (if requested)

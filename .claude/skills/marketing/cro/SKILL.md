# Conversion Rate Optimization (CRO)

Optimize marketing pages for better conversions. Assess pages across seven dimensions, then deliver prioritized recommendations.

## Seven Assessment Dimensions

1. **Value Proposition Clarity** — Can visitors understand the benefit in 5 seconds?
2. **Headline Effectiveness** — Does it communicate core value specifically?
3. **CTA Strategy** — Placement, copy, and hierarchy
4. **Visual Hierarchy** — Scannability and prominence of key elements
5. **Trust Signals** — Social proof and credibility markers
6. **Objection Handling** — FAQ, guarantees, and transparency
7. **Friction Points** — Form fields, navigation, mobile experience

## Output Format

- **Quick Wins**: Changes implementable in under an hour
- **High-Impact Changes**: Significant improvements requiring more effort
- **Test Ideas**: A/B test hypotheses with predicted outcomes
- **Copy Alternatives**: Specific rewrites for underperforming sections

## Required Inputs

- Page URL or description
- Current conversion rate (if available)
- Traffic source
- Primary conversion goal

## Integration

- copywriting: rewriting copy flagged by CRO analysis
- ab-testing: designing tests for high-impact changes
- signup: signup flow specific optimization
- onboarding: post-signup activation optimization
- popups: popup and overlay optimization
- paywalls: in-app upgrade screen optimization

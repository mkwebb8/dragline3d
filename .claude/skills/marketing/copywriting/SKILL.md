# Copywriting

Write clear, compelling marketing copy that drives action. Read `.agents/product-marketing.md` first if it exists.

## What This Skill Covers

- Writing persuasive copy from scratch
- Rewriting weak or unclear sections
- Strengthening headlines, subheadlines, and CTAs
- Improving value propositions and taglines
- Making copy more conversion-focused

## Required Inputs

1. **Page Purpose** — What type of page? What action should visitors take?
2. **Audience** — Who are they? What problem are they solving? What objections do they have?
3. **Product/Offer** — What makes it different? What's the key transformation?
4. **Proof Points** — Stats, testimonials, case studies?
5. **Context** — Where is traffic coming from? What do visitors know before arriving?

## Principles

- Lead with the reader's world, not product features
- Every headline must communicate core value specifically
- One primary CTA per page section
- Proof before promise — establish credibility before asking for action
- Write to one person, not a crowd

## Integration

- product-marketing: foundational context for all copy decisions
- copy-editing: refining and improving existing copy
- cro: conversion optimization of copy elements
- customer-research: customer language to ground copy in reality
